## 📚 Chapters

### 🌐 Phase 1 — Frontend Foundations

> Learn the building blocks of every website on the internet.

<details>
<summary><b>📘 Chapter 01 — HTML Fundamentals</b></summary>
<br/>

- Document structure: `DOCTYPE`, `html`, `head`, `body`
- Headings, paragraphs, links (`a`), images (`img`)
- Lists: ordered (`ol`), unordered (`ul`), definition (`dl`)
- Inline vs block elements
- HTML comments and best practices

</details>

<details>
<summary><b>📘 Chapter 02 — HTML Advanced</b></summary>
<br/>

- Semantic HTML5: `header`, `nav`, `main`, `section`, `article`, `aside`, `footer`
- Forms: all input types, labels, fieldsets, validation attributes
- Multimedia: `audio`, `video`, `iframe`, `canvas` (intro)
- Meta tags and SEO basics
- Accessibility: `aria-label`, `role`, `alt`, tab order
- HTML entities and special characters

</details>

<details>
<summary><b>📘 Chapter 03 — CSS Fundamentals</b></summary>
<br/>

- CSS selectors: element, class (`.`), ID (`#`), attribute, combinators
- Box model: `margin`, `padding`, `border`, `content`, `box-sizing`
- Colors: named, hex, rgb, rgba, hsl
- Typography: `font-family`, `font-size`, `font-weight`, `line-height`, `letter-spacing`
- Units: `px`, `em`, `rem`, `%`, `vw`, `vh`
- `display`, `position`, `z-index`, `overflow`
- CSS specificity and the cascade

</details>

<details>
<summary><b>📘 Chapter 04 — CSS Layouts — Flexbox & Grid</b></summary>
<br/>

- **Flexbox:** `display: flex`, `flex-direction`, `justify-content`, `align-items`, `flex-wrap`, `gap`
- **Flexbox items:** `flex`, `flex-grow`, `flex-shrink`, `flex-basis`, `order`, `align-self`
- **CSS Grid:** `grid-template-columns`, `grid-template-rows`, `grid-area`, `gap`
- Grid auto-placement and `fr` units
- Named grid areas with `grid-template-areas`
- Combining Flexbox and Grid for complex layouts

</details>

<details>
<summary><b>📘 Chapter 05 — CSS Advanced & Animations</b></summary>
<br/>

- Pseudo-classes: `:hover`, `:focus`, `:active`, `:nth-child`, `:not`
- Pseudo-elements: `::before`, `::after`, `::placeholder`
- CSS custom properties (variables): `--color`, `var()`
- Transitions: `transition`, `transition-timing-function`, `transition-delay`
- Animations: `@keyframes`, `animation`, `animation-iteration-count`
- Transform: `translate`, `rotate`, `scale`, `skew`
- CSS filters, `backdrop-filter`, `clip-path`

</details>

<details>
<summary><b>📘 Chapter 06 — Responsive Design</b></summary>
<br/>

- Mobile-first vs desktop-first approach
- Media queries: `@media`, breakpoints
- Responsive images: `srcset`, `picture`, `object-fit`
- Viewport meta tag
- Bootstrap 5: grid system, utility classes, components
- CSS container queries (modern)
- Responsive typography with `clamp()`

</details>

---

### ⚡ Phase 2 — JavaScript

> The language that powers the entire web — frontend and backend.

<details>
<summary><b>📙 Chapter 07 — JavaScript Fundamentals</b></summary>
<br/>

- Variables: `var`, `let`, `const` — differences and best practices
- Data types: `string`, `number`, `boolean`, `null`, `undefined`, `symbol`, `bigint`
- Type coercion and `typeof`
- Operators: arithmetic, comparison (`==` vs `===`), logical, ternary
- Conditions: `if/else`, `switch`
- Loops: `for`, `while`, `do...while`, `for...of`, `for...in`
- Arrays and objects basics

</details>

<details>
<summary><b>📙 Chapter 08 — JavaScript Functions & Scope</b></summary>
<br/>

- Function declarations vs expressions vs arrow functions
- Parameters, default values, rest parameters (`...args`)
- Return values and early returns
- Scope: global, function, block (`let`/`const`)
- Hoisting: variables and functions
- Closures — what they are and why they matter
- IIFE (Immediately Invoked Function Expressions)
- Higher-order functions: intro

</details>

<details>
<summary><b>📙 Chapter 09 — JavaScript DOM & Events</b></summary>
<br/>

- DOM tree and how JavaScript interacts with HTML
- Selecting elements: `querySelector`, `getElementById`, `querySelectorAll`
- Modifying: `innerHTML`, `textContent`, `style`, `classList`
- Creating, inserting, and removing elements
- Event listeners: `addEventListener`, event object, `preventDefault`
- Event bubbling, capturing, and delegation
- Form handling and real-time validation
- `setTimeout`, `setInterval`, `clearInterval`

</details>

<details>
<summary><b>📙 Chapter 10 — JavaScript ES6+ & Modern JS</b></summary>
<br/>

- Destructuring: arrays and objects
- Spread (`...`) and rest operators
- Template literals and tagged templates
- Optional chaining (`?.`) and nullish coalescing (`??`)
- Modules: `import` / `export` (named and default)
- Array methods: `map`, `filter`, `reduce`, `find`, `some`, `every`, `flat`, `flatMap`
- Object methods: `Object.keys()`, `Object.values()`, `Object.entries()`
- Short-circuit evaluation and logical assignment operators

</details>

<details>
<summary><b>📙 Chapter 11 — Asynchronous JavaScript</b></summary>
<br/>

- Synchronous vs asynchronous — the event loop explained
- Callbacks and callback hell (pyramid of doom)
- Promises: creating, chaining `.then()`, `.catch()`, `.finally()`
- `Promise.all`, `Promise.race`, `Promise.allSettled`
- `async` / `await` — clean async code
- Fetch API: making HTTP requests, reading responses
- Error handling in async code with try/catch
- Intro to AJAX and HTTP methods

</details>

<details>
<summary><b>📙 Chapter 12 — JavaScript OOP</b></summary>
<br/>

- Constructor functions and the `new` keyword
- Prototypes and the prototype chain
- ES6 Classes: `class`, `constructor`, methods
- Inheritance with `extends` and `super()`
- Static methods and properties
- Getters and setters
- Private fields (`#field`)
- OOP principles in JavaScript: encapsulation, inheritance, polymorphism

</details>

<details>
<summary><b>📙 Chapter 13 — Browser APIs & Tools</b></summary>
<br/>

- `localStorage`, `sessionStorage`, and `cookies`
- Geolocation API, Clipboard API, Web Notifications
- History API (`pushState`, `popstate`)
- Intersection Observer API (lazy loading)
- Intro to npm, `package.json`, and scripts
- Module bundlers: Vite (setup and usage)
- Chrome DevTools deep dive: console, network, performance

</details>

### ⚛️ Phase 3 — React.js

> Build fast, modern, dynamic UIs with the most popular frontend framework.

<details>
<summary><b>📗 Chapter 14 — React Fundamentals</b></summary>
<br/>

- What is React and the Virtual DOM?
- Setting up a project with Vite
- JSX: writing HTML in JavaScript
- Functional components
- Props: passing and receiving data
- Conditional rendering: `&&`, ternary, if/else
- Rendering lists with `.map()` and `key` prop
- Handling events in React

</details>

<details>
<summary><b>📗 Chapter 15 — React Hooks</b></summary>
<br/>

- `useState` — managing component state
- `useEffect` — side effects, dependencies, cleanup
- `useRef` — referencing DOM elements and persisting values
- `useMemo` — memoizing expensive calculations
- `useCallback` — memoizing functions
- `useContext` — consuming context (intro)
- Rules of Hooks
- Building custom hooks

</details>

<details>
<summary><b>📗 Chapter 16 — React Router</b></summary>
<br/>

- React Router v6 setup
- `<Route>`, `<Routes>`, `<Link>`, `<NavLink>`
- `useParams()` and `useNavigate()`
- Nested routes and `<Outlet>`
- Protected / private routes
- URL search params with `useSearchParams`
- 404 pages and error boundaries in routing
- Lazy loading routes with `React.lazy` and `Suspense`

</details>

<details>
<summary><b>📗 Chapter 17 — State Management</b></summary>
<br/>

- The prop drilling problem and why global state is needed
- Context API: `createContext`, `Provider`, `useContext`
- Redux Toolkit: `createSlice`, `configureStore`
- `useSelector` and `useDispatch`
- Redux Thunk and async actions with `createAsyncThunk`
- RTK Query for data fetching
- Zustand — lightweight alternative to Redux

</details>

<details>
<summary><b>📗 Chapter 18 — React Advanced</b></summary>
<br/>

- TanStack Query (React Query): `useQuery`, `useMutation`, caching
- Form handling with React Hook Form + Zod validation
- Error boundaries for graceful error handling
- Code splitting and `React.lazy` / `Suspense`
- Performance: `React.memo`, `useMemo`, `useCallback`
- Portals for modals and tooltips
- `forwardRef` and the `useImperativeHandle` hook
- React design patterns: compound components, render props

</details>

### 🖥️ Phase 4 — Node.js & Express

> Build powerful servers and RESTful APIs with JavaScript on the backend.

<details>
<summary><b>📒 Chapter 19 — Node.js Fundamentals</b></summary>
<br/>

- What is Node.js and how the event loop works
- Core modules: `fs`, `path`, `os`, `events`, `http`, `url`
- CommonJS (`require`) vs ES Modules (`import`)
- npm: installing packages, `package.json`, `package-lock.json`
- npm scripts and running tasks
- Streams and buffers — working with large data
- Environment variables with `dotenv`
- Building a basic HTTP server from scratch

</details>

<details>
<summary><b>📒 Chapter 20 — Express.js Fundamentals</b></summary>
<br/>

- Setting up an Express server
- Routing: `GET`, `POST`, `PUT`, `DELETE`, `PATCH`
- Route parameters (`req.params`) and query strings (`req.query`)
- Request body: `req.body`, `express.json()` middleware
- Middleware: built-in, third-party (`morgan`, `cors`), custom
- Router — organizing routes in separate files
- Serving static files with `express.static`
- Error handling middleware (4-argument handler)

</details>

<details>
<summary><b>📒 Chapter 21 — REST APIs</b></summary>
<br/>

- REST principles: stateless, client-server, uniform interface
- HTTP methods and their correct usage
- HTTP status codes: 2xx, 3xx, 4xx, 5xx
- Designing clean RESTful API endpoints
- CRUD operations and resource naming conventions
- Request validation with `express-validator` or `zod`
- API versioning (`/api/v1/`)
- Testing APIs with Postman — collections, environments

</details>

<details>
<summary><b>📒 Chapter 22 — Authentication & Security</b></summary>
<br/>

- Authentication vs authorization
- Password hashing with `bcrypt`
- JWT (JSON Web Tokens): creating, signing, verifying
- Auth middleware for protected routes
- Refresh tokens and token expiry
- CORS setup with `cors` package
- Rate limiting with `express-rate-limit`
- Security headers with `helmet.js`
- Input sanitization and SQL/NoSQL injection prevention

</details>

### 🍃 Phase 5 — MongoDB

> Master the most popular NoSQL database for modern web applications.

<details>
<summary><b>📓 Chapter 23 — MongoDB Fundamentals</b></summary>
<br/>

- NoSQL vs SQL — when to use MongoDB
- MongoDB Atlas setup and connecting to the cloud
- Collections, documents, and BSON
- CRUD operations: `insertOne`, `insertMany`, `find`, `findOne`, `updateOne`, `updateMany`, `deleteOne`, `deleteMany`
- Query operators: `$eq`, `$gt`, `$lt`, `$in`, `$and`, `$or`
- Projection, sorting, limiting, and skipping
- Aggregation pipeline: `$match`, `$group`, `$sort`, `$project`, `$lookup`
- Indexes and performance optimization

</details>

<details>
<summary><b>📓 Chapter 24 — Mongoose ODM</b></summary>
<br/>

- Connecting Mongoose to MongoDB Atlas
- Defining schemas: types, required, default, unique
- Creating and using models
- CRUD with Mongoose methods
- Schema validation: built-in and custom validators
- Relationships: referencing with `ObjectId` and `populate()`
- Embedded documents (subdocuments)
- Virtuals and computed properties
- Middleware: `pre` and `post` hooks (e.g., hash password before save)
- Pagination patterns

</details>

## 🔥 How to Use

<div align="center">

```
Step 1 → Follow the phases in order — never skip
Step 2 → Read the PDF notes for the chapter carefully
Step 3 → Close the notes and attempt the practice set solo
Step 4 → Stuck? Struggle for 15 more minutes first
Step 5 → Check solutions and understand WHY it works
Step 6 → Build the mini project after each phase
Step 7 → Move forward. Repeat. Ship something. 🚀
```

</div>

<br/>

> 💡 **Pro Tip:** Type every solution yourself. Copy-paste builds nothing.

> ⚠️ **Do not skip phases.** Every chapter builds directly on the last.

> 🔁 **Revise before moving on.** Shaky foundations = confused future you.

> 🛠️ **Build the projects.** Theory without projects is incomplete.
